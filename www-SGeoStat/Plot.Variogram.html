plot.variogram()

Description:
Plot empirical variogram estimates, optionally plotting a fitted variogram model.

Usage:
plot(variogram.obj, variogram.model.obj)

Required arguments:
variogram.obj: a variogram object generated by variogram()

Optional arguments:
variogram.model.obj: a variogram model object generated by a model fitting routine.

Value:
NULL

Examples:
> plot(stations.v)
> plot(stations, stations.v.m)
