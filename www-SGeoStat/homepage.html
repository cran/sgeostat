An Object-oriented Framework for Geostatistical Modeling in S+

James J. Majure
Iowa State University
Geographic Information Systems
Support and Research Facility

This document describes an object-oriented framework to support geostatistical modeling within the S+ computing environment. Like most classical statistical packages, S+ has no resident spatial statistical capabilities. The framework described in this document makes use of S+'s flexible, object-oriented programming environment to implement a set of exploratory and modeling tools for geostatistical data. The term "framework" is used because the tools are easily extensible by defining additional functions that operate on the proposed data structures. The tools described provide basic capabilities necessary for geostatistical modeling while still allowing access to the S+ environment for any traditional or non-standard data analysis needs.
