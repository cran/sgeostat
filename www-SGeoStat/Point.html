point()

Description:
Create an object of class point from a data frame.

Usage:
point(data.frame, x='x', y='y')

Required arguments:
data.frame: a data frame containing the x and y coordinates for each point and the variables observed at each point

Optional arguments:
x: the name of the column in data.frame that contains the x coordinate
y: the name of the column in data.frame that contains the y coordinate

Value:
A point object

Examples:
> stations <- point(dataf)
> stations <- point(dataf,x='Xcoord',y='Ycoord')
