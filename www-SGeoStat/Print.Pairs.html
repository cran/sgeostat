print.pairs()

Description:
Print descriptive information about a pairs object.

Usage:
print.pairs(pairs.obj)

Required arguments:
pairs.obj: a pairs object generated by pairs()

Value:
NULL

Examples:
> stations.prs

Pairs object: stations.prs

Type: isotropic
Number of pairs: 10387
Number of lags: 15
Max h: 299959.515
