Geostatistics for S+: Downloading and Installation

Click here (SGeoStat.tar.Z) to download the programs. They arrive in the form of a compressed unix tar file (tape archive file). On a unix platform, use the following commands to uncompress and untar them: 


% uncompress SGeoStat.tar.Z
% tar -xvf SGeoStat.tar

This will put several ascii files into the current working directory. These files must each be "sourced" into S+. Before sourcing the files, however, global change the string: "/home/majure/majure/variograms/.Functions" to the name of the directory in which you want to place the S+ functions.

To use the programs in S+, you must "attach" the directory in which you put the function, using the attach() function.
